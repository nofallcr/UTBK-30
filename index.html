Bank Rumus 📐
Semua rumus penting UTBK Matematika dalam satu halaman.
Kuadrat Sempurna
(a+b)² = a²+2ab+b² (a-b)² = a²-2ab+b² (a+b)(a-b) = a²-b²
Rumus ABC
ax²+bx+c = 0 x = (-b ± √(b²-4ac)) / 2a D = b²-4ac
D>0: 2 akar beda · D=0: kembar · D<0: tidak real
Vieta (Hub. Akar)
x₁+x₂ = -b/a x₁·x₂ = c/a (x₁-x₂)² = (x₁+x₂)²-4x₁x₂
SPLDV
ax+by = c dx+ey = f → Substitusi / Eliminasi
Eliminasi cepat: samakan koefisien salah satu variabel
Barisan & Deret
Aritmetika: Uₙ = a+(n-1)b · Sₙ = n/2·(2a+(n-1)b) Geometri: Uₙ = a·rⁿ⁻¹ · Sₙ = a(rⁿ-1)/(r-1) · S∞ = a/(1-r) jika |r|<1
Komposisi Fungsi
(f∘g)(x) = f(g(x)) (g∘f)(x) = g(f(x)) Perhatian: f∘g ≠ g∘f
Fungsi Invers
y=f(x) → tukar x,y → selesaikan y (f⁻¹∘f)(x) = x (f∘f⁻¹)(x) = x
Fungsi Kuadrat
f(x) = ax²+bx+c Puncak: (-b/2a, -D/4a) Sumbu simetri: x = -b/2a
a>0: parabola ke atas (min) · a<0: ke bawah (maks)
Domain Penting
√f(x) : f(x) ≥ 0 1/f(x) : f(x) ≠ 0 log f(x): f(x) > 0
Logaritma
aˣ=b ↔ ᵃlog b=x · log(ab)=loga+logb · log(a/b)=loga-logb log aⁿ=n·loga · ᵃlog b = logb/loga (ganti basis)
Ukuran Pemusatan
Mean = Σxᵢ/n Median = nilai tengah (terurut) Modus = nilai terbanyak
Ukuran Penyebaran
Ragam σ² = Σ(xᵢ-x̄)²/n Simp. baku σ = √σ² Jangkauan = maks-min
Mean Berkelompok
x̄ = Σ(fᵢ·xᵢ)/Σfᵢ xᵢ = titik tengah kelas
Kuartil
Q1=25%, Q2=50%, Q3=75% IQR = Q3-Q1
Median & Modus Berkelompok
Median = L+((n/2-F)/f)×p · Modus = L+(d₁/(d₁+d₂))×p L=tepi bawah · F=frek kum. sebelum · f=frek kelas · p=panjang kelas
Peluang Dasar
P(A) = n(A)/n(S) 0 ≤ P(A) ≤ 1 P(A') = 1-P(A)
Gabungan
P(A∪B) = P(A)+P(B)-P(A∩B) Saling lepas: P(A∩B)=0 Saling bebas: P(A∩B)=P(A)·P(B)
Permutasi
P(n,r) = n!/(n-r)! Siklis = (n-1)! Berulang = nʳ
Urutan diperhitungkan → Permutasi
Kombinasi
C(n,r) = n!/(r!(n-r)!) C(n,r) = C(n,n-r)
Urutan tidak diperhitungkan → Kombinasi
Segitiga
Luas = ½×alas×tinggi Heron: L=√(s(s-a)(s-b)(s-c)), s=(a+b+c)/2 Pytagoras: c²=a²+b²
Lingkaran
L=πr² · K=2πr Busur=(θ/360°)×2πr Juring=(θ/360°)×πr²
Volume Bangun Ruang
Kubus=s³ · Balok=p×l×t Tabung=πr²t · Kerucut=⅓πr²t Bola=⁴⁄₃πr³ · Limas=⅓×Lalas×t
Koordinat & Gradien
d=√((x₂-x₁)²+(y₂-y₁)²) M=((x₁+x₂)/2,(y₁+y₂)/2) m=(y₂-y₁)/(x₂-x₁)
Perbandingan Dasar
sin=depan/miring (SOH) cos=samping/miring (CAH) tan=depan/samping (TOA)
Identitas Utama
sin²x+cos²x = 1 1+tan²x = sec²x tan x = sin x/cos x
Sudut Istimewa
0° 30° 45° 60° 90° sin: 0 ½ ½√2 ½√3 1 cos: 1 ½√3 ½√2 ½ 0 tan: 0 ⅓√3 1 √3 ∞
Jumlah Sudut
sin(A±B)=sinA cosB±cosA sinB cos(A±B)=cosA cosB∓sinA sinB tan(A±B)=(tanA±tanB)/(1∓tanA tanB)
Sudut Ganda & Aturan Sinus/Cosinus
sin2A=2sinA cosA · cos2A=cos²A-sin²A=1-2sin²A a/sinA=b/sinB=c/sinC · a²=b²+c²-2bc cosA
Turunan Dasar
d/dx[c]=0 · d/dx[xⁿ]=nxⁿ⁻¹ d/dx[sin x]=cos x · d/dx[cos x]=-sin x d/dx[eˣ]=eˣ · d/dx[ln x]=1/x
Aturan Turunan
Rantai: [f(g(x))]'=f'(g(x))·g'(x) Hasil kali: (uv)'=u'v+uv' Pembagian: (u/v)'=(u'v-uv')/v²
Integral Dasar
∫xⁿdx=xⁿ⁺¹/(n+1)+C · ∫eˣdx=eˣ+C ∫sinx dx=-cosx+C · ∫cosx dx=sinx+C ∫1/x dx=ln|x|+C
Nilai Ekstrem
Stasioner: f'(x)=0 Maks jika f''(x)<0 Min jika f''(x)>0
FPB & KPK
FPB×KPK = a×b Cara: faktorisasi prima
Aritmetika Sosial
Untung=HJ-HB · Rugi=HB-HJ %Untung=(Untung/HB)×100% Bunga=P×r×t
Perpangkatan
aᵐ×aⁿ=aᵐ⁺ⁿ · aᵐ/aⁿ=aᵐ⁻ⁿ (aᵐ)ⁿ=aᵐⁿ · a⁻ⁿ=1/aⁿ a^(1/n)=ⁿ√a
Rasio & Proporsi
a:b=c:d → ad=bc Senilai: y/x=konstan Berbalik nilai: xy=konstan
